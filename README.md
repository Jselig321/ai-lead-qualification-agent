# AI Lead Qualification Agent

An agentic AI system that researches companies, evaluates lead quality, and scores sales prospects based on Ideal Customer Profile (ICP) fit. The app uses a multi-step pipeline that scrapes company websites, analyzes business context, and categorizes leads into suitability tiers.

Built as a lightweight demo of how AI can automate sales qualification workflows.

---

## Features

- CSV batch lead scoring
- Single company lookup mode
- Automated website scraping
- Structured ICP scoring (1–10)
- Lead categorization (Very Suitable / Moderate / Low Match)
- Sorted results leaderboard
- Clean dashboard UI
- JSON parsing + structured reasoning
- Streamlit interactive interface

---

## Tech Stack

- Python
- LangChain
- OpenAI GPT models
- Streamlit
- pandas
- BeautifulSoup
- requests

---

## How It Works

1. Upload a CSV of leads or enter a single company
2. The agent scrapes the company website
3. AI evaluates business context
4. The system scores ICP fit
5. Leads are categorized and ranked

This simulates a real revenue operations pipeline used for automated prospect qualification.
